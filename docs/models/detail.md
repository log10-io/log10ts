# Detail

Specifies the detail level of the image. Learn more in the [Vision guide](/docs/guides/vision/low-or-high-fidelity-image-understanding).

## Example Usage

```typescript
import { Detail } from "log10ts";

let value: Detail = Detail.Low;
```

## Values

| Name   | Value  |
| ------ | ------ |
| `Auto` | auto   |
| `Low`  | low    |
| `High` | high   |
